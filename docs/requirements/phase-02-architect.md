## 方块配置系统设计

```typescript
interface BlockConfig {
  id: number;              // 唯一标识
  name: string;            // 显示名称
  texture: string;         // 纹理路径/图集坐标
  hardness: number;        // 硬度（挖掘时间 = hardness / 工具效率）
  transparent: boolean;    // 是否透明（影响渲染和相邻面剔除）
  collision: AABB | null;  // 碰撞箱，null = 无碰撞（如花草）
  toolRequired?: ToolType; // 需要特定工具才能掉落
  drops?: DropConfig[];    // 破坏后掉落物
}
```
