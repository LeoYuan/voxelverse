## 活塞推动规则

1. 活塞只能推动非空气、非基岩、非黑曜石的方块
2. 一次性推动的方块链长度 ≤ 12
3. 如果方块链末端被阻挡（实体方块），活塞不激活
4. 粘性活塞在信号消失后，拉回最前方的一个方块
5. 被推动的方块如果携带红石信号，信号在移动过程中中断
