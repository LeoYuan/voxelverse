# VoxelVerse

浏览器中的体素沙盒世界。当前主玩法是轻松向的「基地战役」：持续建设同一座基地，完成章节目标，再守过有上限的夜袭。

## 在线体验

```bash
npm install
npm run dev
```

打开浏览器访问 `http://localhost:5173`，选择模式后进入游戏。若 5173 已被占用，可通过 `npm run dev -- --port 4177` 使用其他端口。

## 技术栈

- **渲染**: Three.js + InstancedMesh（单 Chunk 一个 Draw Call）
- **语言**: TypeScript，Vite 构建
- **地形**: 自定义 3D Perlin 噪声，确定性生成
- **测试**: Vitest（221 项单元测试）+ Playwright（13 项端到端测试）

## 已实现功能

### 基地战役（默认模式）

- 5 章持续基地战役：建造避难所、完善生产设施、加固防线并坚守黎明
- 每章由「白天建设 → 主动开始守夜 → 限时击杀 → 领取物资」组成
- 夜袭敌人数量有上限；失败后回到准备阶段，基地、背包和章节进度保留
- 左键可近战攻击僵尸、骷髅、爬行者和动物，工具会提高伤害
- 第 5 章后进入逐步增强的无尽守夜
- 单槽自动存档保存章节、玩家与重生点、背包、奖励队列及玩家方块改动
- 重新打开页面可继续战役；菜单中的「重新开始」需要二次点击确认
- 区块数据和网格分帧流式加载，避免首屏同步生成大范围世界

### 其他模式

- **闯关建造**：20 关结构目标，适合短局练习
- **自由建造**：创造模式与无限方块
- **自由生存**：连续昼夜、环境刷怪、采集与合成；不使用基地战役存档

### Phase 1 — 创世（已完成）
- 16x16x16 Chunk 流式加载/卸载
- 基于 InstancedMesh 的批量体素渲染，同屏 10 万+ 方块 60 FPS
- 第一人称相机 + WASD 飞行/行走
- 射线检测：左键破坏、右键放置方块
- 3D Perlin 噪声地形（草地/泥土/石头三层）
- 天空盒 + 方向光 + 阴影

### Phase 2 — 造物主之手（已完成）
- 30+ 种方块（草、土、石、木、玻璃、楼梯、台阶、矿石等）
- 方块配置系统：ID、颜色、硬度、透明度、碰撞箱
- 掉落物实体（受重力、可被收集）
- 屏幕准星 + 指向方块名称提示
- 破坏/放置音效

### Phase 3 — 荒野契约（进行中）
- **生存模式**: 重力、跳跃、摔落伤害、20 点生命值/饥饿值
- **怪物**: 僵尸（近战）、骷髅（射箭）、爬行者（靠近爆炸）
- **动物**: 牛，白天在草地上生成
- **昼夜循环**: 20 分钟/天，夜晚怪物生成
- **光照系统**: 天空光照 + 夜晚怪物生成阈值
- **工具系统**: 木/石/铁/钻石镐，耐久度与挖掘加速
- **床与重生点**: 右键床设置重生点，死亡后按 R 重生
- **合成系统**: 2x2 随身合成 + 3x3 工作台配方（ planks、工具台、熔炉、箱子等）
- **红石矿石**: Y<16 地下生成，挖掘掉落红石粉

### Phase 4 — 地下脉动（进行中）
- **红石引擎**: 每 Tick BFS 信号传播，双缓冲状态机
- **红石粉**: 相邻连接，信号每格衰减 -1（最大 15）
- **红石火把**: NOT 门逻辑，输入为 0 时输出 15
- **电源**: 拉杆（切换）、按钮（1 秒脉冲）、红石块（常亮）
- **红石灯**: 信号 >= 1 时自动亮起
- **中继器**: 1-4 tick 延迟，信号放大至 15
- **逻辑门验证**: 非门、或门、与非门单元测试通过

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 空格 | 跳跃 / 上升（创造模式）|
| Shift | 加速 / 下降 |
| 鼠标左键 | 攻击准星内生物；无目标时破坏方块 |
| 鼠标右键 | 放置方块 / 交互（拉杆、床）|
| 数字键 1-9 | 选择快捷栏 |
| F | 达成章节建设目标后开始守夜 |
| C | 打开/关闭合成界面 |
| H | 打开模式与设置菜单 |
| G | 在自由建造/自由生存之间切换（战役中禁用）|
| E | 进食 |
| 创造面板 | 选择红石元件、工具和全部建筑方块 |
| R | 死亡后重生 |

## 项目结构

```
src/
  blocks/          方块注册表与定义
  engine/          Chunk、地形生成、昼夜循环
  player/          玩家控制器、生命值/饥饿值系统
  rendering/       VoxelRenderer（InstancedMesh）
  redstone/        红石引擎与信号传播
  crafting/        合成配方系统
  campaign/        章节状态机、目标、波次、存档与 HUD
  combat/          玩家攻击选择、伤害和冷却
  entities/        掉落物、怪物、动物
  utils/           Vec3、AABB、Perlin 噪声
  tests/           单元测试（Vitest）
e2e/               浏览器端到端测试（Playwright）
```
