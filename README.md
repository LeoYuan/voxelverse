# VoxelVerse

浏览器中的体素沙盒世界，类 Minecraft 的 3D 建造与生存体验。

## 在线体验

```bash
npm install
npm run dev
```

打开浏览器访问 `http://localhost:5173`，点击画面锁定鼠标即可开始游戏。

## 技术栈

- **渲染**: Three.js + InstancedMesh（单 Chunk 一个 Draw Call）
- **语言**: TypeScript，Vite 构建
- **地形**: 自定义 3D Perlin 噪声，确定性生成
- **测试**: Vitest，76+ 单元测试

## 已实现功能

### Phase 1 — 创世（已完成）
- 16x16x16 Chunk 流式加载/卸载
- 基于 InstancedMesh 的批量体素渲染，同屏 10 万+ 方块 60 FPS
- 第一人称相机 + WASD 飞行/行走
- 射线检测：左键破坏、右键放置方块
- 3D Perlin 噪声地形（草地/泥土/石头三层）
- 天空盒 + 方向光 + 阴影

### Phase 2 — 造物主之手（已完成）
- 30+ 种方块（草、土、石、木、玻璃、楼梯、台阶、矿石等）
- 方块配置系统：ID、颜色、硬度、透明度、碰撞箱
- 掉落物实体（受重力、可被收集）
- 屏幕准星 + 指向方块名称提示
- 破坏/放置音效

### Phase 3 — 荒野契约（进行中）
- **生存模式**: 重力、跳跃、摔落伤害、20 点生命值/饥饿值
- **怪物**: 僵尸（近战）、骷髅（射箭）、爬行者（靠近爆炸）
- **动物**: 牛、猪、羊，白天在草地上生成
- **昼夜循环**: 20 分钟/天，夜晚怪物生成
- **光照系统**: 天空光照 + 夜晚怪物生成阈值
- **工具系统**: 木/石/铁/钻石镐，耐久度与挖掘加速
- **床与重生点**: 右键床设置重生点，死亡后按 R 重生
- **合成系统**: 2x2 随身合成 + 3x3 工作台配方（ planks、工具台、熔炉、箱子等）
- **红石矿石**: Y<16 地下生成，挖掘掉落红石粉

### Phase 4 — 地下脉动（进行中）
- **红石引擎**: 每 Tick BFS 信号传播，双缓冲状态机
- **红石粉**: 相邻连接，信号每格衰减 -1（最大 15）
- **红石火把**: NOT 门逻辑，输入为 0 时输出 15
- **电源**: 拉杆（切换）、按钮（1 秒脉冲）、红石块（常亮）
- **红石灯**: 信号 >= 1 时自动亮起
- **中继器**: 1-4 tick 延迟，信号放大至 15
- **逻辑门验证**: 非门、或门、与非门单元测试通过

## 操作说明

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
| 空格 | 跳跃 / 上升（创造模式）|
| Shift | 加速 / 下降 |
| 鼠标左键 | 破坏方块 |
| 鼠标右键 | 放置方块 / 交互（拉杆、床）|
| 数字键 0-9 | 选择方块 |
| G | 切换创造/生存模式 |
| E | 进食 |
| N/M/T/Y/U/I/O | 红石粉/火把/灯/拉杆/按钮/红石块/中继器 |
| Z/X/C/V | 木/石/铁/钻石镐 |
| R | 死亡后重生 |

## 项目结构

```
src/
  blocks/          方块注册表与定义
  engine/          Chunk、地形生成、昼夜循环
  player/          玩家控制器、生命值/饥饿值系统
  rendering/       VoxelRenderer（InstancedMesh）
  redstone/        红石引擎与信号传播
  crafting/        合成配方系统
  entities/        掉落物、怪物、动物
  utils/           Vec3、AABB、Perlin 噪声
  tests/           单元测试（Vitest）
```
